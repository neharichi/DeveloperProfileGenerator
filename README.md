# Developer Profile Generator

## README
This is a command line application that generates a profile for Github user.
application asks user to input their GitHub user name, and choose one of 4 predefined colors. 

## Command to run application
node index.js

### modules used

Inquirer, Util, Axios, Puppeteer

# Modify to Run
For  puppeteer function to work, a path to the html file to be converted needs to be specified. Currently, it is set to my local repository. please change it to your local path where index.html
